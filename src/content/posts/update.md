# 网站更新日志

本文记录了博客的每一次更新。

---

## 2026-04-25

### 新增功能

1. **文章修订历史功能**
   - 自动从 Git 仓库提取文章修改历史
   - 在文章页面底部显示修订次数、修改时间和提交信息
   - 点击 commit hash 可跳转到 GitHub 查看详细修改
   - 自动计算文章发布和最后更新时间，提醒读者文章新鲜度
2. **文章大纲功能优化**
   - 自动生成结构化目录大纲
   - 支持滚动时自动高亮当前章节
   - 点击大纲项可快速跳转到对应内容位置
   - 大纲固定在页面右侧，跟随滚动保持在可视区域
3. **部署流程优化**
   - 新增 `pnpm update-diff` 命令用于更新文章修订历史
   - 完善部署文档，明确部署前必须执行的命令
3. **移除了 GitHub Actions 部署支持**
   - 将 GitHub Actions 部署转为 Netlify 部署
4. **修改主域名119567.xyz的解析逻辑**
   - 使用海内外分流解析逻辑
   - 增加国内esa进行域名解析，海外eo进行域名解析，加快各地区速度
   - **主域名blym.top仍保持不变，并无后续分流计划**

## 2026-02-25

### 新增功能

- **GitHub Actions 自动部署**
   - 配置了 GitHub Actions 自动构建部署
   - 推送到 main 分支自动构建部署到 GitHub Pages
   - 支持绑定自定义域名

2. **Netlify 部署支持**
   - 添加 netlify.toml 配置文件
   - 支持一键部署到 Netlify

3. **域名切换组件**
   - 侧边栏显示当前访问域名
   - 显示多个域名：主站（blog.blym.top、blog.119567.xyz）和备用站（hengduan.dpdns.org、hengduan.de5.net）
   - 实时显示各域名的网络延迟
   - 当前访问的域名自动高亮显示
   - 点击在当前标签页切换域名

4. **不蒜子统计接入**
   - 侧边栏 Profile 区域显示今日访问量和今日访客数
   - 页脚显示今日访问、今日访客、总访问、总访客

5. **移动端布局优化**
   - 侧边栏功能移至移动端顶部

6. **分类排序优化**
   - 未分类分类默认放在最底部
   - 其他分类按字母顺序排列

### 功能优化

1. **主题设置面板优化**
   - 主题切换按钮添加背景模糊效果
   - 彩虹模式支持实时调节颜色变换速率
   - 主题色滑块实时生效
   - 统一按钮点击状态视觉反馈

## 2026-02-24

### 新增功能

1. **评论系统**
   - 集成了 Giscus 评论系统
   - 支持亮/暗主题切换
   - 位于文章末尾

2. **文章末尾固定内容**
   - GitHub 编辑链接
   - 文章修订历史

3. **侧边栏 Profile 组件升级**
   - 头像改为圆形样式
   - 显示文章统计数据（文章数、总字数）
   - 集成社交平台链接入口（GitHub、Gitee、Bilibili）

4. **文章更新检测功能**
   - 右下角小铃铛组件
   - 自动检测 RSS 订阅，发现新文章或更新时通知
   - 支持查看文章变更内容（DIFF 显示）

5. **文章置顶功能**
   - 支持在文章 frontmatter 中添加 `pinned: true` 置顶文章
   - 置顶文章显示"置顶"标签

6. **主题切换同步修复**
   - 修复了主题切换功能与亮/暗主题切换不同步的问题
   - 建立了单一的主题状态源

7. **默认主题色修改**
   - 默认主题色从 250（青色）改为 120（绿色）
